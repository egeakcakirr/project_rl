Task: Build the smallest complete runnable Actor-Critic reinforcement learning script.

Core requirements:

* Standard library only.
* Standalone script.
* 1D grid world with states 0..9.
* Start at 0, goal at 9.
* Two actions: left and right.
* Clamp invalid boundary movement.
* Reward +10 at goal and -1 otherwise.
* Terminate at goal or max_steps=25.

Actor-Critic requirements:

* Actor: tabular policy preferences.
* Critic: tabular value estimates.
* Use TD error.
* Training uses softmax action selection.
* Evaluation uses greedy policy.

Required execution:

* Train for 40 episodes.
* Evaluate for 5 episodes.
* Print reward, steps, and termination reason.
* Write generation_report.csv in the current directory.
* Include a textual policy summary.
* Include random.seed(42).
* Include if **name** == "**main**".
* Use type hints.
* No third-party packages.
* No infinite loops.
