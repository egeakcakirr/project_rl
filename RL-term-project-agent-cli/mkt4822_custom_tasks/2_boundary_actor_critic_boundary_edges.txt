Task: Build an Actor-Critic RL program focused on boundary and edge-case handling.

Environment:

* 1D grid with states 0..9.
* Start at state 0.
* Goal at state 9.
* Action 0 moves left.
* Action 1 moves right.
* At state 0, action 0 must keep the agent at state 0.
* At state 9, action 1 must keep the agent at state 9 until the episode ends.
* Reward is +10 when first reaching state 9 in an episode and -1 otherwise.
* Episode terminates at goal or max_steps=35.

Actor-Critic:

* Tabular actor with preferences for each state/action.
* Tabular critic with state values.
* Softmax action selection during training.
* TD error updates both actor and critic.

Output requirements:

* Train for 100 episodes.
* Evaluate for 10 greedy episodes.
* Print reward, steps, and termination reason for every training episode.
* Print evaluation results.
* Generate generation_report.csv with episode, total_reward, steps, termination.
* Include textual policy and value summaries.
* Use random.seed(42).
* Use type hints.
* Include if **name** == "**main**".
* Standard library only.
* No infinite loops.
