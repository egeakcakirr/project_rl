Task: Build a compact but complete Actor-Critic RL application.

Environment:

* 1D grid world with states 0..9.
* Start state is 0.
* Goal state is 9.
* Actions: 0 = left, 1 = right.
* Clamp boundary transitions.
* Reward: +10 at goal, -1 otherwise.
* Episode ends at goal or max_steps=25.

Actor-Critic:

* Tabular actor with state-action preferences.
* Tabular critic with state-value estimates.
* TD error updates both actor and critic.
* Softmax action selection during training.

Training:

* Train for only 30 episodes.
* Every episode must terminate safely.
* Print reward and steps for every episode.

Evaluation:

* Run 5 greedy evaluation episodes.
* Print evaluation rewards and steps.

Outputs:

* generation_report.csv with episode, total_reward, steps, termination.
* Simple textual visualization of learned policy.
* random.seed(42).
* if **name** == "**main**".
* Type hints.
* Standard library only.
* No placeholders.
