Task: Build a full Actor-Critic reinforcement learning application with extended training.

Environment specification:

* 1D grid with states from 0 to 14.
* Start state is 0.
* Goal state is 14.
* Actions are 0 = left and 1 = right.
* Clamp movement to valid boundaries.
* Reward is +15 at the goal and -1 otherwise.
* Episode ends at goal or max_steps=60.

Actor-Critic specification:

* Use a tabular actor with policy preferences.
* Use a tabular critic with value estimates.
* Use softmax action selection during training.
* Use temporal-difference error.
* Update both actor and critic each step.
* Use standard library only.

Training:

* Train for 300 episodes.
* Print summary for every episode.
* Enforce max_steps in every loop.

Evaluation:

* Run 20 greedy evaluation episodes.
* Print average evaluation reward and average steps.

Outputs:

* generation_report.csv with episode, total_reward, steps, termination.
* Textual visualization of policy direction for each state.
* Textual visualization of value estimates.
* random.seed(42).
* if **name** == "**main**".
* Modern type hints.
* No placeholders.
