Task: Build a runnable Actor-Critic RL application that produces multiple textual learning summaries.

Environment:

* 1D grid world with states 0..9.
* Start at 0.
* Goal at 9.
* Actions are left and right.
* Clamp boundary transitions.
* Reward is +10 at the goal and -1 otherwise.
* Episode terminates at goal or max_steps=50.

Actor-Critic:

* Use tabular actor preferences.
* Use tabular critic values.
* Use softmax exploration for training.
* Use TD error update.
* Use greedy evaluation after training.

Training and evaluation:

* Train for 200 episodes.
* Evaluate for 15 episodes.
* Print per-episode training summary.
* Print evaluation summary.

Visualization and reporting:

* Print learned policy as arrows or action names.
* Print critic state values.
* Print recent reward trend using text only.
* Write generation_report.csv with episode, total_reward, steps, termination.
* Use relative path only.
* Include random.seed(42).
* Include type hints.
* Include if **name** == "**main**".
* Standard library only.
* No incomplete sections.
