Task: Build a reproducible standalone Actor-Critic reinforcement learning program.

Environment specification:

* Use a deterministic 1D grid world with states 0..9.
* Start at state 0 and goal is state 9.
* Actions are left and right.
* Clamp all transitions to valid state boundaries.
* Reward is +10 at the goal and -1 otherwise.
* Terminate each episode at goal or max_steps=40.

Actor-Critic specification:

* Use tabular policy preferences for the actor.
* Use tabular state values for the critic.
* Use TD error for critic and actor updates.
* Use softmax exploration during training.
* Use greedy action selection during evaluation.

Reproducibility requirements:

* Use random.seed(42).
* Keep all randomness controlled through the standard random module.
* Do not depend on external packages or external files.

Required output:

* Train for 120 episodes.
* Evaluate for 10 episodes.
* Print per-episode training reward, steps, and termination reason.
* Generate generation_report.csv with episode, total_reward, steps, termination.
* Print a final textual policy summary.
* Include if **name** == "**main**".
* Use type hints.
* Standard library only.
