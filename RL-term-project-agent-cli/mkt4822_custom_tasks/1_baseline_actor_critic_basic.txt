Task: Build a runnable Actor-Critic reinforcement learning application for a simple 1D grid world.

Environment specification:

* States are integers from 0 to 9.
* Start state is 0.
* Goal state is 9.
* Actions are 0 = move left and 1 = move right.
* State transitions must be clamped to the range [0, 9].
* Reward is +10 when the goal is reached and -1 otherwise.
* Each episode ends when the goal is reached or max_steps=35 is exceeded.

Actor-Critic specification:

* Use a tabular actor represented by policy preferences for each state-action pair.
* Use a tabular critic represented by state-value estimates.
* Use softmax action selection during training.
* Use temporal-difference error to update both actor and critic.
* Use Python standard library only.

Training and evaluation:

* Train for 100 episodes.
* Each episode must have a hard max_steps guard.
* Evaluate for 10 greedy episodes after training.
* Print total reward, steps, and termination reason for every training episode.
* Print evaluation results clearly.

Output requirements:

* Generate generation_report.csv in the current working directory using a relative path.
* CSV columns: episode, total_reward, steps, termination.
* Include a simple textual visualization of learned policy and value estimates.
* Include random.seed(42).
* Include modern Python type hints.
* Include if **name** == "**main**".
* Do not use third-party packages.
* Do not leave placeholders or incomplete sections.
