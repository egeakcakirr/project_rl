Task: Build a runnable Actor-Critic application where negative rewards are common.

Environment:

* 1D grid with states 0..9.
* Start at 0 and goal at 9.
* Actions: left or right.
* Clamp boundary transitions.
* Reward is +10 at goal.
* Reward is -2 for every non-goal step.
* Episode ends at goal or max_steps=35.

Actor-Critic:

* Tabular actor preferences.
* Tabular critic values.
* Softmax action selection.
* TD error update.
* Standard library only.

Training:

* Train for 120 episodes.
* Print every episode's total reward, step count, and termination reason.
* Enforce max_steps strictly.

Evaluation:

* Run 10 greedy episodes.
* Print evaluation total rewards and average performance.

Outputs:

* generation_report.csv in current working directory.
* CSV columns: episode, total_reward, steps, termination.
* Textual policy and value visualization.
* random.seed(42).
* if **name** == "**main**".
* Type hints.
* No third-party packages.
